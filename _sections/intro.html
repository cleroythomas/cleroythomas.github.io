Welcome to the Reduce Austin Homelessness website

This is an application based effort to address the Homelessness issue in Austin, Texas. This effort is based on 3 separate but connected (by dataset) mobile / desktop apps. 1.Homeless Services Locator. 2.Homeless Activity Reporter. 3.Austin Homeless Activity Dashboard(dashboard for the Austin Homelessness Reporter)
