Welcome to the Reduce Austin Homelessness website

This is an application based effort to address the Homelessness issue in Austin, Texas. This effort is based on 3 separate but connected (by dataset) mobile / desktop apps. Homeless Services Locator and Homeless Activity Reporter and Dashboard.
